Supplementary file 4.

Differential gene expression data of control vs MIA samples, and WGCNA gene module assignment at E12.5. Columns indicate gene symbols (gene_name), log 2 fold change (LogFC), log 2 counts per million (LogCPM), likelihood ratio statistics (LR), p value (PValue), false discovery rate adjusted p value (FDR), and WGCNA module assignment (moduleColors and moduleColors_merged). A total of 16,613 genes were tested.
