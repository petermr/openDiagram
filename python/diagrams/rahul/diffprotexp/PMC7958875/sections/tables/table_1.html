We analyzed three different data sets. First, data from the Malaria Cell Atlas ( Howick et al., 2019) consisting of single cell P. berghei ookinetes collected from pooled An.stephensi midguts was compared to a data set collected simultaneously but from An. coluzzii midguts that we called Pb vector species data set. Second, we analyzed single cell P. bergheiookinetes that were collected from four individual An. coluzzii midguts and named this data set Pb single gut data set. Third, we used P. falciparum bulk RNA-seq data from a previously published data set ( Dahalan et al., 2019) that investigated P. falciparum infection in An. coluzzii in mated compared to virgin females. We named this data set Pf bulk data set.
