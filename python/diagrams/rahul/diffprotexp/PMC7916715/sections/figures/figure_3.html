Differential expression of LE genes in PCs and mPCs compared to prostate tissues and primary PCs, respectively. ( A) The analyses were performed using the GEPIA2 database. T: tumor (PC), N: normal prostate tissues, and TPM: transcripts per million. Statistical analyses were performed by GEPIA2, * p < 0.05. ( B) Analyses were performed using the Sawyers dataset in R2: Genomics Analysis and Visualization Platform. Expressions were presented as log2-transformed data.
